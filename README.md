# Lista de Livros Mentores e Exercícios de Python

Este repositório reúne dois temas principais:

- uma análise de livros recomendados por CEOs e líderes globais;
- uma coleção de exercícios de lógica, estruturas de dados e fundamentos de programação em Python.

## 📚 Análise de livros mentores

Este projeto busca identificar padrões de conhecimento e os pontos de convergência intelectual entre líderes globais e influenciadores de referência, como Elon Musk, Bill Gates e Barack Obama. Através de técnicas de web scraping e análise de dados, é possível descobrir quais livros formam a base intelectual compartilhada por essas personalidades.

### Tecnologias utilizadas

- Python
- BeautifulSoup
- Pandas

### Estrutura do projeto

- scraper.py: coleta dos dados nas URLs selecionadas.
- analise_consenso.py: análise e identificação de padrões.
- base_recomendacoes_ceos.csv: base bruta com as recomendações.
- livros_ouro_consenso.csv: dataset refinado com os livros de maior autoridade.

### Como executar

1. Instale as dependências: pip install pandas requests beautifulsoup4
2. Execute o scraper: python scraper.py
3. Rode a análise: python analise_consenso.py
